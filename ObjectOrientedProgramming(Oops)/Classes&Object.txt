Classes and Objects:-

Objects are the real word entity. For Example:Watch,Pen,Animals etc.


Encapsulation: Encapsulation is defined as the wrapping up of data & methods under a single unit.It also implements data hiding.

Construtor:Constructor is a special method which is invoked automatically at the time of object creation.


-Constructor have the same name as class or structure.

-Constructor do not have a return type .(Not even void)

-Constructors are only called once at object creation.

-Memory allocation happens when constructor is called.
